# Проект автоматизации тестирования Stellar Burgers

# Описание
Проект содержит автотесты для проверки функциональности сайта Stellar Burgers (https://stellarburgers.education-services.ru/)

# Тестируемый функционал
- Регистрация пользователя
- Вход в систему (4 способа)
- Переход в личный кабинет
- Переход из личного кабинета в конструктор
- Выход из аккаунта
- Работа разделов конструктора

# Технологии
- Python 3
- Selenium WebDriver
- PyTest
- Faker (генерация тестовых данных)
